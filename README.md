# Python Projects

A collection of beginner-level Python projects built to strengthen my understanding of core programming fundamentals.

## Projects Focus
- User input handling
- Conditional statements
- Loops
- Basic functions
- Simple program logic
- Clean terminal interaction

## 📂 Projects Included

### 1. Calculator
A simple command-line calculator that performs basic arithmetic operations.
- Addition, Subtraction, Multiplication, Division
- Concepts: `if / elif / else`, floating-point numbers, basic arithmetic

### 2. Number Guessing Game
A random number guessing game where the user tries to guess a number between 1 and 100.
- Concepts: `while` loops, random number generation, user feedback logic, loop control flow

### 3. Shopping App
A basic program that calculates the total cost of items based on price and quantity.
- Concepts: type conversion (`int`, `float`), f-strings, simple multiplication logic

### 4. Weight Converter
A simple weight conversion tool between kilograms and pounds.
- Concepts: conditional logic, unit conversion, input validation basics

### 5. Rock Paper Scissors
A command-line Rock Paper Scissors game against the computer with ASCII art.
- Concepts: random module, conditionals, input validation, try/except

### 6. Ticket Calculator
A theme park ticket pricing system based on height and age.
- Concepts: nested conditionals, input handling, billing logic

### 7. Pizza Price Calculator
A pizza ordering system with size and topping pricing.
- Concepts: nested conditionals, `.upper()` input handling, pricing logic

### 8. Tip Calculator
A bill splitting calculator with tip percentage.
- Concepts: float formatting, arithmetic, f-strings
