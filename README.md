# python-beginner-projects

This repository contains a collection of beginner-level Python projects that I built to strengthen my understanding of core programming fundamentals.

These projects focus on:

- User input handling
- Conditional statements
- Loops
- Basic functions
- Simple program logic
- Clean terminal interaction

## 📂 Projects Included

### 1. Calculator
A simple command-line calculator that performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

Concepts practiced:
- Conditional statements (`if / elif / else`)
- Floating-point numbers
- Basic arithmetic operations

---

### 2. Number Guessing Game
A random number guessing game where the user tries to guess a number between 1 and 100.

Concepts practiced:
- `while` loops
- Random number generation
- User feedback logic
- Loop control flow

---

### 3. Shopping App
A basic program that calculates the total cost of items based on price and quantity.

Concepts practiced:
- Type conversion (`int`, `float`)
- String formatting (f-strings)
- Simple multiplication logic

---

### 4. Weight Converter
A simple weight conversion tool between kilograms and pounds.

Concepts practiced:
- Conditional logic
- Unit conversion
- Input validation basics

---
